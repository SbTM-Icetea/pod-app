⚙️ Konfigurasi Label Pengiriman

Hubungkan Google Sheets sebagai sumber data label.

1 Nama Perusahaan

2 Google Sheets

Ambil dari URL Sheets kamu:
docs.google.com/spreadsheets/d/[ID INI]/edit
Pastikan Sheets di-share Anyone with the link → Viewer.

Nama tab di bagian bawah spreadsheet kamu.

Buat di Google Cloud Console → Credentials → Create API Key → aktifkan Google Sheets API.
API Key disimpan hanya di browser kamu, tidak dikirim ke server lain.
